# 🌌 DevSpace Browser Extension

A premium, highly-customizable Chrome/Edge New Tab extension crafted for developers, designers, and power users. DevSpace transforms your standard new tab page into a beautiful, functional, and highly efficient dashboard.

## ✨ Key Features

*   **🔍 Pro Search Bar:** A sleek, macOS Spotlight-inspired search bar with deep space dark-mode aesthetics, cinematic neon inner-glow, and an interactive 3D Three.js rotating sun animation.
*   **⏱️ Real-Time Celestial Clock Complication:** A stunning analog and digital clock widget situated in the bottom left, featuring a fully functional 24-hour sun and moon track. The glowing sun or moon accurately orbits the clock face based on the current time of day in real-time.
*   **🤖 Floating AI Assistant Hub:** Quick access to the world's most powerful AI tools organized by category (Chat, Image, Code). 
    *   *Features 100% premium, glowing SVG/FontAwesome vector iconography with brand-specific hover gradients.*
    *   *Includes: ChatGPT, Claude, Gemini, Copilot, Grok, Midjourney, DALL-E, GitHub Copilot, Cursor, and more!*
*   **🌤️ Pro-Level Live Weather:** Top-left ambient weather display fetching real-time data complete with dynamic, animated weather conditions.
*   **💎 Glassmorphism & Neon Aesthetics:** The entire interface is built on cutting-edge web design principles featuring heavy use of `backdrop-filter` blurring, soft gradient borders, and reactive hover animations.
*   **🔒 Built-In Dev Protections:** Custom keyboard shortcut handlers and right-click protections.

## 🚀 Installation & Setup

1. Clone or download this repository to your local machine:
   ```bash
   git clone https://github.com/Abhisahu143/DevSpace2.git
   ```
2. Open your chromium-based browser and navigate to the Extensions page (`chrome://extensions/` or `edge://extensions/`).
3. Enable **Developer Mode** using the toggle switch in the top right corner.
4. Click on the **Load unpacked** button.
5. Select the folder containing the DevSpace extension files.
6. Open a new tab and enjoy your new dashboard!

## 🛠️ Built With
*   **HTML5 / CSS3 / Vanilla JavaScript** - Lightweight and incredibly fast.
*   **Three.js** - For the fully-rendered 3D interactive sun element inside the search bar.
*   **FontAwesome** - For high-quality, scalable vector icons.
*   **WeatherAPI** - Supplying real-time, accurate global weather data.
